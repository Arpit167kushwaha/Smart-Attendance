# Smart Attendance System

A Smart Attendance Management System developed to automate and simplify attendance tracking. The system enables users to mark attendance efficiently using QR Code technology and provides real-time attendance records through a user-friendly interface.

---

## Features

### Attendance Management

* Automated Attendance Tracking
* QR Code Based Attendance System
* Real-Time Attendance Recording
* Attendance History Management

### User Features

* Secure User Registration
* User Login Authentication
* Profile Management
* Attendance Monitoring

### Admin Features

* Manage Users
* View Attendance Records
* Monitor Attendance Statistics
* Generate Attendance Reports

## Installation

### Clone Repository

```bash

```

### Backend Setup

```bash
mvn spring-boot:run
```

### Frontend Setup

```bash
npm install
npm start
```
